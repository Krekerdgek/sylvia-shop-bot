# Шрифты для Sylvia Bot

В этой папке должны находиться файлы шрифтов для генерации визиток.

## Рекомендуемые шрифты

- `Inter-Regular.ttf` - основной шрифт
- `Inter-Bold.ttf` - жирное начертание

## Где скачать

Шрифты Inter можно скачать бесплатно с официального сайта:
https://rsms.me/inter/

Или используйте системные шрифты, но для совместимости лучше загрузить свои.
