# Шаблоны визиток для Sylvia Bot

В этой папке должны находиться файлы шаблонов визиток в формате PNG.

## Требования к шаблонам

- Размер: **800×600 пикселей**
- Формат: PNG с прозрачностью (RGBA)
- Ориентация: горизонтальная

## Структура файлов

- `template_1.png` - Базовый шаблон (бесплатный)
- `template_2.png` - Премиум шаблон (50 ⭐)
- `template_3.png` - Wildberries стиль (100 ⭐)

## Рекомендации по дизайну

Оставьте свободное место для:
- QR-кода (обычно слева или по центру)
- Текста благодарности
- Названия магазина

## Пример размещения элементов
────────────────────────────────┐
│ │
│ [QR-код] Спасибо за покупку│
│ [300x300] Название магазина │
│ Артикул: 12345678 │
│ │
│ Отсканируйте для отзыва │
└────────────────────────────────┘

## Создание шаблонов

Вы можете создать шаблоны в любом графическом редакторе (Photoshop, Figma, Canva) и сохранить их сюда.
